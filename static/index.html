DOCUMENTACION

CATEGORIAS

GET

/category

-> Obtiene todas las categoria existentes

Ejemplo:

https://localhost:3002/category

Respuesta:

{ _id: "ad2o3r9udsf029fjdsf", name: "Limpieza" }

GET

/category/:id

-> Encuentra la categoria que corresponda al ID pasado por parametro

Ejemplo:

https://localhost:3002/category/ad2o3r9udsf029fjdsf

Respuesta:

{ _id: "ad2o3r9udsf029fjdsf", name: "Limpieza" }

POST

/category

-> Agrega una nueva categoria

Ejemplo:

https://localhost:3002/category

Respuesta:

{ ok: true, msg: 'Se creo la categoria exitosamente'}
